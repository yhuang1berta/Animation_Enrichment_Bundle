Animation Enrichment Bundle (AEB) Documentation

(Note: all property name used to initialize the library can be accessed directly, example of accessing color argument: lib.color)

Only ONE animation can be run on a webpage at a time!

AEB instance

Property Name Default value Type Description
canvas null html canvas object "The canvas object wish to be used to play the animation on."
r 5 int "The radius of the points on canvas."
border_width 5 int "The width of the canvas' border (if any)."
after providing border_width, AEB will set the canvas' border to this width with solid black color!
color "black" string "The color of the points."
"If color is "random", then all points' color would be randomized (as shown in the introduction page)"
(Note: if color passed in is not a valid css color string, the color would be set to "black" by default.)
filled false boolean "Whether to fill the points with current color."
horizontal_spacing 10 int/float "The spacing between each point horizontally in terms of its radius."
vertical_spacing 15 int/float "The spacing between each point vertically in terms of its radius."

AEB example

const c = document.createElement("canvas")
// create an instance of AEB library with canvas c, filled with points
// with radius of 6 pixels, 5 pixels border, and default spacing between the points
const lib = AEB(c, 6, 5, "blue", true, 10, 15)

textFall Animation

Property Name Default value Type Description
text "TEXT FALL" string "The text wish to be rendered on canvas." All letters are Captialized at the moment!
(note: use "/" to switch line in the input text)
x_modifier 100 int/float "The starting x coordinate of the top left point."
y_modifier 100 int/float "The starting y coordinate of the top left point."

textFall example

const c = document.createElement("canvas")
const lib = AEB(c, 6, "blue", true, 10, 15)
// add Text Fall animation on canvas c, points will plot out the word "Text Fall"
// top left point starts at (50, 150)
lib.textFall("text fall", 50, 150)

// Check out Text Fall demo in header to see effect!

disperse Animation

Property Name Default value Type Description
text "DISPERSE" string "The text wish to be rendered on canvas." All letters are Captialized at the moment!
(note: use "/" to switch line in the input text)
x_modifier 100 int/float "The starting x coordinate of the top left point."
y_modifier 100 int/float "The starting y coordinate of the top left point."
cursor_range 100 int/float "The range of points that will be affected as the cursor moves, the larger the number, the wider the effect range."
return_speed 15 int/float "The speed that controls how fast the points would return to their original position, the larger the number, the slower they return."
noise 0.5 float "The noise argument controls how much fluctuation the points will make, the larger the number, the more vibrant the points will be."
random_start false boolean "Whether the starting position will be randomized, if false, the points will start at their desired locatoin."

disperse example

const c = document.createElement("canvas")
const lib = AEB(c, 6, "blue", true, 10, 15)
// add disperse animation on canvas c, points will plot out the word "DISPERSE"
// top left point starts at (70, 150), cursor's range at 150px
// return speed at 20, and noise factor of 0.5, points start at their original location
lib.disperse("disperse", 70, 150, 150, 20, 0.5, false)

// Check out Disperse demo in header to see effect!
